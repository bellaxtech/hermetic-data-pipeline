Hermetic Data Pipeline — End-to-End Architecture

Data Ingestion

  • • Scrapy 기반 문서 크롤링

  • • CDC 기반 증분 적재 (updated_at)

  • • Small Files Compaction

RAG AI Service

  • • LangChain 프롬프트 템플릿
  • • FastAPI SSE 스트리밍
